Welcome!

Welcome to Springies.jl! This is a personal sandbox project for having fun with dynamical systems problems. It's a wrapper for OrdinaryDiffEq.jl.

Quickstart guide

Dynamical systems are encoded in Springies.jl using Springy types, which contain the system parameters and external forcing. Each Springy type must have a corresponding differentials! function, which describes how the differential terms in the system are computed.

We then solve the system with the springy_solve function.

Check out the Example Gallery for project ideas, and the API Reference for a more detailed breakdown of functionality. Have fun!
